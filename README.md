#### 编码规范

缩进:  四个空格

大括号：

​	1.大括号不可省略

​	2.开放大括号的位置和语句放在同一行

命名:

​	1.构造函数：大驼峰式命名法

​	2.函数和方法名：小驼峰式命名法

​	3.变量名：全部小写，单词以下划线分隔

​	4.常量：全部大写，单词以下划线分隔

​	5.属性名：小驼峰式命名法

​	6.私有属性：一个下划线 + 小驼峰式命名法

​	7.私有方法：一个下划线 + 小驼峰式命名法
